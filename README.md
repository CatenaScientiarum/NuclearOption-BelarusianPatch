# Nuclear Option - Belarusian Localization Patch

Belarusian (Беларуская) translation mod for Nuclear Option.

## Installation
1. Install [BepInEx](https://github.com/BepInEx/BepInEx/releases) (Unity IL2CPP)
2. Place all files in `BepInEx/plugins/LocalizationPatch_blr/`:
   - `LocalizationPatch.dll`
   - `blr.json`
   - `Exo2-Regular.ttf`
3. Launch the game
